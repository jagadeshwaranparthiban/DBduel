Instructions for the Quiz

1. Quiz Format:

  • The quiz consists of 15 multiple-choice questions across different topics in Core Java .
  • Each question has four options, labeled A, B, C, and D. Only one option is correct.

2. Topics Covered:

  • Comparison Operators
  • Modifiers and Keywords
  • Data Structures
  • Polymorphism
  • Exceptions Handling
  • Code Snippets and Output
  • Platform Independence and Object-Oriented Concepts

3. Answering the Questions:

  • Read each question carefully before selecting your answer.
  • Choose the most appropriate option for each question.

4. Scoring:

  • Each correct answer earns 1 point.
  • There is no negative marking for incorrect answers.

5. Time Limit:

  • The quiz has a time limit of 20 minutes.
  • Make sure to submit your answers before the time runs out.

6. Submission:

  • Once you have answered all the questions, click the "Submit" button.
  • Your score will be displayed on the next page.

7. Additional Guidelines:

  • This quiz is an individual effort; collaboration with others is not permitted.
  • No use of external resources (e.g., internet, books, or notes) is not allowed.
  • Ensure you understand the concepts before attempting the quiz.

8. Results:

  • Your score will be shared after the quiz.
